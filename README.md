# username-platform-checkers
Checking for avaiable usernames on various platforms.

<h3>How to use</h3> First of all, you have to had Node installed.

Execute <b>install.bat</b> to install npm packages, edit <b>config.json</b> with your api's key, and then open <b>start.bat</b>.
Choose the platform you want to check, make sure your user/id files be like this, for example:

![alt text](https://github.com/retrohv4/username-platform-checkers/blob/main/img/exemple.png)

Software will automatically save avaiable usernames on <b>logs.txt</b>.
If you find out any bugs, please report on [Issues](https://github.com/retrohv4/username-platform-checkers/issues) tab.
